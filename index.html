iframe-embeddable pet

给你的页面 养一只 会走的桌宠

canvas-pet 是一个纯前端的桌面宠物组件：会走、会睡、会被鼠标吓跑，还会偶尔冒一句金句气泡。一行 iframe 即可嵌入任何网页。

查看实时演示 GitHub

特性

零依赖、零构建。一份 JS + 几张精灵图，往页面里一扔就能跑。

🪶

一行嵌入

任意网页加一个 iframe 即可，不需要改你的技术栈或打包流程。

🐾

自然行为

随机在走 / 呆 / 睡之间切换，带呼吸、眨眼、影子、边界弹性。

🖱️

可拖拽

鼠标按住宠物拖动，松手带惯性抛出。靠近它会受惊逃跑。

💬

金句气泡

定时冒一句文案，文案在配置里随意改。

🧩

可换皮

替换 assets/pet/ 里的精灵图 + 改配置，就是一只新宠物。

🪟

点击穿透

全屏覆盖模式下，iframe 不会挡住页面交互，只在命中宠物时接管。

实时演示

下面是一个定尺寸的 iframe，宠物在框内自由活动。把鼠标移过去试试——它会躲开你，你也可以按住它拖动。

提示：金句气泡约每 9–18 秒冒一次；鼠标靠近 92px 内会触发逃跑。

两种嵌入方式

最简单的是「盒子嵌入」——给 iframe 一个固定尺寸，宠物就在里面活动。要让它满屏游走又不挡页面，用「全屏覆盖」。

① 盒子嵌入（最简单） 
<!-- 把 src 指向你部署后的 pet.html 路径 -->
<iframe src="https://your-username.github.io/canvas-pet/pet.html"
        width="320" height="240" frameborder="0"
        style="border:0;background:transparent;"
        title="canvas-pet"></iframe>
② 全屏覆盖 + 点击穿透（带拖拽） 
// 宿主页面：让 iframe 满屏覆盖、默认鼠标穿透，仅在命中宠物时接管拖拽。
// source 必须与 pet.config.js 的 parentSource 一致（默认 'pet-host'）。

<iframe id="petFrame" src="https://your-username.github.io/canvas-pet/pet.html"
        style="position:fixed;inset:0;width:100vw;height:100vh;border:0;z-index:900;pointer-events:none;background:transparent;"></iframe>

<script>
(function () {
  const SOURCE = 'pet-host';
  const frame = document.getElementById('petFrame');
  let hit = null, dragging = false;
  const inHit = (x, y) => hit && x >= hit.left && x <= hit.right && y >= hit.top && y <= hit.bottom;

  window.addEventListener('pointermove', e => frame.contentWindow.postMessage(
    { source: SOURCE, type: 'douknow-pet-pointer', x: e.clientX, y: e.clientY }, '*'), { passive: true });

  window.addEventListener('pointerdown', e => {
    if (!inHit(e.clientX, e.clientY)) return;
    dragging = true;
    frame.style.pointerEvents = 'auto';
    frame.contentWindow.postMessage(
      { source: SOURCE, type: 'douknow-pet-drag-start', x: e.clientX, y: e.clientY }, '*');
  });
  window.addEventListener('pointermove', e => {
    if (!dragging) return;
    frame.contentWindow.postMessage(
      { source: SOURCE, type: 'douknow-pet-drag-move', x: e.clientX, y: e.clientY }, '*');
  });
  const end = () => {
    if (!dragging) return;
    dragging = false;
    frame.style.pointerEvents = '';
    frame.contentWindow.postMessage({ source: SOURCE, type: 'douknow-pet-drag-end' }, '*');
  };
  window.addEventListener('pointerup', end);
  window.addEventListener('pointercancel', end);

  window.addEventListener('message', e => {
    const d = e.data || {};
    if (d.source === 'douknow-pet' && d.type === 'douknow-pet-bounds') hit = d.bounds;
  });
})();
</script>

完整的消息协议字段表见 docs/embed.md。

本地预览

因为是 iframe + 相对路径，需要通过 HTTP 访问（直接双击 html 打开也行，但部分浏览器会限制 file:// 协议）。

# 在项目根目录任选一种
npx serve .
# 或
npx http-server -p 3000

三步自定义

  1. 改金句 / 尺寸 / 速度 / 数量：打开 pet.config.js，编辑里面的字段。
  2. 换皮肤：替换 assets/pet/ 下的图片（命名见 docs/reskin.md）。
  3. 部署：推到 GitHub，仓库 Settings → Pages → 从 main 分支发布（根目录）。
